## 📋 Features

### Core Requirements Implemented:
- **Dashboard Overview**
  - Summary Cards: Total Balance, Income, Expenses
  - Balance Trend Line Chart (Last 6 Months)
  - Spending Breakdown Doughnut Chart

- **Transactions Section**
  - Full transactions table with Date, Description, Category, Amount, Type
  - Search functionality
  - Category filter
  - Add new transaction (Admin only)

- **Role-Based UI**
  - **Viewer Role**: Can only view data (Add/Edit/Delete buttons hidden)
  - **Admin Role**: Can Add, Edit & Delete transactions
  - Role switcher in sidebar

- **Insights Section**
  - Highest spending category
  - Monthly comparison insight

- **Additional Features**
  - Fully Responsive design (Mobile + Desktop)
  - Clean and modern UI with cards and shadows
  - Modal for adding/editing transactions
  - Smooth interactions

## 🛠️ Technologies Used
- HTML5
- CSS3 (Custom styling with Flexbox & Grid)
- Vanilla JavaScript
- Chart.js (for charts)
- Font Awesome Icons

## 🚀 How to Run Locally

1. Repository ko clone karo ya download karo
2. Folder mein `index.html` file ko double-click karke browser mein kholo
3. No installation required
